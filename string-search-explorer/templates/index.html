Algorithm Complexity Reference
Algorithm Best Case Average Case Worst Case Space Key Idea
Naive (Brute Force) O(n) O(n * m) O(n * m) O(1) Slide pattern one position at a time, compare left to right
Rabin-Karp O(n + m) O(n + m) O(n * m) O(1) Rolling hash to skip windows; verify on hash match
Knuth-Morris-Pratt O(n) O(n + m) O(n + m) O(m) LPS failure table avoids re-comparing matched prefix
Boyer-Moore O(n / m) O(n / m) O(n * m) O(alphabet) Right-to-left scan; bad character heuristic for large skips
